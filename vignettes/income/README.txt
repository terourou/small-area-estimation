
Code to model the relationship between income, age, and qualification,
in order to estimate income in the whole population.

To run the code, use 'make' at the command line.

The data is a 'synthetic unit record file' (SURF) retrieved from https://www.stats.govt.nz/services/customised-data-services/statistics-for-university-staff-and-students/new-zealand-income-survey-super-surf/ on 8 February 2021.
